# Sales Intelligence Ultra Dashboard

## Project Overview
This project is an interactive sales intelligence dashboard designed to analyze sales performance across brands, regions, retailers, and time periods.

## Objective
The objective was to create a lightweight, browser-based dashboard that presents sales data in a visually engaging and business-friendly way.

## Key Features
- Monthly revenue visualization
- Brand-wise revenue comparison
- Regional sales share analysis
- Retailer unit performance
- Interactive year and region filters
- Intelligence summary for quick interpretation
- Responsive dashboard interface

## Tools and Technologies
- HTML5
- CSS3
- JavaScript
- Chart.js
- GitHub Pages
- AI Prompt Engineering

## What I Learned
Through this project, I practiced dashboard design, business intelligence thinking, front-end development, and prompt engineering for analytics-focused problem solving.
